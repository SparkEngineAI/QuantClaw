# QuantClaw GitHub Pages Site

This folder is a publish-ready GitHub Pages bundle.

## Included Files

- `index.html`: main site entry
- `figs/`: images used by the page
- `QuantClaw_demo.mp4`: demo video
- `.nojekyll`: disables Jekyll processing on GitHub Pages

## Deploy

### Publish from repository root

1. Copy all files in this folder to the repository root.
2. In GitHub repository settings, open `Pages`.
3. Set source to `Deploy from a branch`.
4. Choose your branch and `/ (root)`.

### Publish from `docs/`

1. Copy this folder into your repository as `docs/`.
2. In GitHub repository settings, open `Pages`.
3. Set source to `Deploy from a branch`.
4. Choose your branch and `/docs`.

## Notes

- All local assets are referenced with relative paths.
- Replace the placeholder `GitHub Link` and `Paper Link` in `index.html` before publishing if needed.
